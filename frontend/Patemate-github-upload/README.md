# PetPulse 智能遛狗健康管家网页原型

这是根据项目创意制作的单页网页原型，补充文档仅作为页面结构和功能参考。

当前版本：

- 顶部导航切换引导、今日、散步、记录、档案、提醒页面。
- 引导页恢复产品主图，作为用户进入页面的第一屏。
- 今日页只保留用户关心的状态、建议和提醒。
- 散步页整合轨迹、时长、心率、消耗、距离和速度曲线。
- 记录页保留每日散步记录和每日便便记录。
- 档案页支持多只狗狗的新增、切换和删除。
- 健康助手页通过本地代理连接 DeepSeek，未配置 key 时会使用本地模拟回答。
- 数据使用合理模拟值，点击刷新数据会更新面板和曲线。

普通预览可直接打开 `index.html`。如果要使用健康助手接口，请用本地服务运行：

```bash
DEEPSEEK_API_KEY=你的密钥 node server.mjs
```

然后访问 `http://127.0.0.1:4173/`。
